Como rodar o projeto:

Obs: Ter instalado o Rails, Ruby e Sqlite3

1 - Baixar o projeto
2 - No terminal de comando rodar o rails server na pasta do projeto
3 - Acessar a url com a rota /items essa rota é para adicionar novos itens e listar os cadastrados
4 - Na rota de items/new é para adicionar o arquivo que será salvo tanto 
localmente na pasta uploads e no banco de dados.
